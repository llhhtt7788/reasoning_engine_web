## 📖 界面导览

### 左侧：Session 管理区
```
┌─────────────────┐
│  + 新建对话     │  ← 创建新的对话
├─────────────────┤
│ [当前对话] 🔵   │  ← 当前激活的对话（蓝色高亮）
│ 如何治疗糖尿病？ │
│ 2小时前 · 8条   │
├─────────────────┤
│ 昨天的对话      │  ← 历史对话（点击切换）
│ 1天前 · 12条    │
├─────────────────┤
│ ...             │
├─────────────────┤
│ 📎 上传文件     │  ← 上传知识文档
│ 📋 查看上传列表 │  ← 查看所有上传的文件
└─────────────────┘
```

### 中间：聊天主区
- 显示当前对话的所有消息
- 底部输入框：输入问题并发送
- 顶部工具栏：
  - Conversation ID 显示
  - Reset 按钮（重置为默认 user_id）
  - 🔍 调试按钮（手动打开调试面板）

### 右侧：智能调试面板
- **自动展开**：当 AI 使用推理（Reasoning）功能时自动滑出
- **手动控制**：点击顶部"🔍 调试"按钮可随时打开/关闭
- **三个 Tab**：
  - 🧠 **思路**：查看 AI 的推理过程
  - 📊 **上下文**：查看上下文选择、Token 使用等
  - 🛤️ **路径**：查看 LangGraph 决策路径

## 🚀 快速开始

### 1️⃣ 开始新对话
1. 点击左上角"**+ 新建对话**"按钮
2. 在底部输入框输入您的问题
3. 按 Enter 或点击发送按钮

### 2️⃣ 查看对话历史
- 左侧列表显示所有历史对话
- 当前对话显示蓝色边框高亮
- 点击任意历史对话可快速切换

### 3️⃣ 删除不需要的对话
1. 鼠标悬停在某个对话上
2. 右侧出现 ❌ 删除按钮
3. 点击后弹出确认窗口
4. 确认删除

### 4️⃣ 上传知识文档
**方式一：点击上传**
1. 点击左下角"**📎 上传文件**"按钮
2. 选择文件（支持 PDF、TXT、MD、DOC、DOCX）
3. 等待上传成功提示

**方式二：拖拽上传（推荐）**
1. 直接将文件拖到左侧 Session 列表区域
2. 看到蓝色边框高亮时松开鼠标
3. 自动开始上传

### 5️⃣ 查看上传的文件
1. 点击左下角"**📋 查看上传列表**"按钮
2. 弹出窗口显示所有上传的文档
3. 可以查看状态、大小、入库进度等

### 6️⃣ 查看 AI 推理过程
- **自动展开**：当 AI 使用深度推理时，右侧面板自动滑出
- **手动打开**：点击顶部"**🔍 调试**"按钮
- **切换 Tab**：点击"思路"、"上下文"、"路径"查看不同信息
- **关闭面板**：点击右上角 ❌ 按钮

## 💡 使用技巧

### ✨ Session 自动保存
- 每次发送消息后，对话自动保存到本地
- 刷新页面不会丢失任何内容
- 关闭浏览器后重新打开，历史对话依然存在

### ✨ 智能面板展开
- 简单问题（如"你好"）不会触发推理，面板保持隐藏
- 复杂问题（如"如何治疗 2 型糖尿病？"）会触发深度推理，面板自动展开
- 您可以随时手动控制面板的显示/隐藏

### ✨ 快速切换对话
- 当前对话始终显示在列表顶部
- 其余对话按最近活动时间排序
- 点击即可无缝切换，无需等待

### ✨ 拖拽上传更便捷
- 无需点击按钮，直接拖拽文件到左侧栏
- 支持从文件管理器、桌面直接拖拽
- 上传状态实时显示（上传中、成功、失败）

## 🛠️ 常见问题

### Q: 我的历史对话保存在哪里？
A: 保存在浏览器的 localStorage 中，只在当前浏览器可见。未来版本会支持云端同步。

### Q: 可以同时打开多个对话吗？
A: 当前版本一次只能查看一个对话。切换对话时，当前对话会自动保存。

### Q: 删除对话后可以恢复吗？
A: 删除是永久性的，无法恢复。删除前会弹出二次确认窗口。

### Q: 右侧面板为什么有时不自动展开？
A: 只有当 AI 使用推理（Reasoning）功能时才会自动展开。简单问题通常不需要推理，因此面板保持隐藏。您可以随时手动打开查看上下文信息。

### Q: 上传的文件有大小限制吗？
A: 建议单个文件不超过 10MB。大文件可能导致解析时间较长。

### Q: 支持哪些文件格式？
A: 当前支持：
- PDF（文本型，非扫描件）
- TXT、MD（纯文本、Markdown）
- DOC、DOCX（Word 文档）
