# w.2.4.0 前端 UX 优化

## 变更记录

### 1. 问答滚动体验优化 (MessageList)

**目标**：解决问答过程中，上一轮问题遮挡当前回答的问题；确保每一轮问答开始时，提问能丝滑置顶，且回答内容有足够的展示空间。

**实现方案**：
- **User 提问时**：强制触发 `scrollIntoView({ block: 'start' })`，将用户问题滚动至视口顶部。
- **Assistant 回复/流式输出时**：
  - 移除自动滚动到底部（Scroll to Bottom）的逻辑。
  - 避免因自动滚底导致位于顶部的“用户问题”被顶出视口。
  - 允许用户手动控制滚动，或静默让回答内容向下填充。

**效果**：
- 用户发送问题后，问题固定在顶部。
- AI 回答在下方流式生成，视口保持稳定，不发生跳动。
